### Setup
Put file "tetris.db" into "bin" folder.

### Controls
<ul>
  <li>Up arrow - rotate the shape</li>
  <li>Down arrow - move shape down</li>
  <li>Left arrow - move shape left</li>
  <li>Right arrow - move shape right</li>
</ul>
